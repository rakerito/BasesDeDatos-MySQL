# Bases de Datos en MySQL

## Descripción
Este repositorio contiene una colección de **trabajos académicos** realizados en la materia de **Bases de Datos** en la Universidad Tecnológica de San Juan del Río.  
Cada archivo `.sql` incluye la creación de tablas, relaciones y restricciones, mostrando prácticas de modelado de datos relacionales en **MySQL Workbench**.

## Contenido
- **ABASTECIMIENTO (Inciso A, B, C)** → Variantes de un modelo de abastecimiento con diferentes estructuras.
- **Agente de seguros** → Gestión de agentes y sus relaciones con clientes y pólizas.
- **Club náutico** → Administración de socios, actividades y recursos de un club.
- **Concesionario** → Control de vehículos, clientes y transacciones de ventas.
- **Entidad bancaria** → Modela entidades financieras, sucursales, vigilantes, jueces y atracadores.
- **Holding empresarial** → Representación de empresas y sus vínculos corporativos.
- **Hospitales 2** → Gestión de hospitales, pacientes y servicios médicos.

## Cómo usar
# 1. Clonar este repositorio
git clone https://github.com/rakerito/BasesDeDatos-MysQL.git
# 2. Abrir MySQL Workbench
# 3. Importar el archivo .sql deseado
# 4. Ejecutar los scripts para crear las tablas y relaciones
# 5. Probar consultas sobre las tablas creadas
